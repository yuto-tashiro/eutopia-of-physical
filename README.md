# 身体のユートピア (Eutopia of Physical)

**情報に埋もれた身体を、問い直す**  
*Beyond data, beyond objectivity—Rediscovering what true health means.*

---

## プロジェクト概要 (Project Overview)

「身体のユートピア」は、AI時代における身体性の価値を再発見し、健康×文化×身体性を融合した新しい健康観を提唱するプロジェクトです。

理学療法士・博士（人間健康科学）の田代雄斗が主宰し、科学的エビデンスと感性を融合させながら、スポーツ、文化、アート、ビジネスなど多岐にわたる領域で活動しています。

---

## コンセプト (Concept)

### なぜ今、身体か

1. **AI時代の身体性**: 情報が溢れる現代だからこそ、代替不可能な「身体」の価値が際立ちます
2. **科学と感性の融合**: スポーツから始まった身体への探求を、文化・アートへの興味とともに深めています
3. **Utopia から Eutopia へ**: 理想郷ではなく、実践を通じて「良き場所」を今ここに創ります

---

## 活動の3つの柱 (Three Pillars)

### 🔬 Science & Education (科学と教育)
エビデンスに基づいた身体の理解を深め、専門家・一般の方々へ向けた教育活動を展開

### 🎨 Culture & Art (文化と芸術)
日本文化に根ざした健康観を再認識するとともに、世界各国の文化に合わせた健康観を理解。質的アプローチとアートの融合を探求

### 🤝 Social & Well-being (社会とウェルビーイング)
コミュニティ形成支援、企業の健康経営サポート、ソーシャルデザインを通じて、誰もが「整う」環境を創造

## 技術スタック (Tech Stack)

- **HTML5**: セマンティックな構造
- **CSS3**: 伝統的な日本の色彩（生成り、墨色、朱色など）を使用したデザインシステム
- **JavaScript**: スクロールアニメーション、Intersection Observer API
- **フォント**: Noto Serif JP (日本語) / Cormorant Garamond (英語)

## ローカル環境での実行 (Local Development)

1. リポジトリをクローン:
```bash
git clone https://github.com/yuto-tashiro/eutopia-of-physical.git
cd eutopia-of-physical
```

2. ブラウザで `index.html` を開く

## GitHub Pages でのデプロイ (Deploy with GitHub Pages)

1. リポジトリの Settings に移動
2. Pages セクションで Source を `main` ブランチに設定
3. `https://yuto-tashiro.github.io/eutopia-of-physical/` でアクセス可能になります
